## Estruturas de Decisão em R: If, Else If e Else

A base da lógica de programação reside na capacidade de um software tomar decisões. Em R, assim como em outras linguagens, o fluxo de execução de um script é controlado por **condições**. Estas estruturas permitem que seu código se comporte de maneira diferente dependendo dos dados que ele processa.

## A Instrução `if`
A instrução `if` é o pilar fundamental do controle de fluxo. Ela define um bloco de código que será executado **apenas se** a condição fornecida for avaliada como `TRUE`.

```r
a <- 33
b <- 200

if (b > a) {
  print("b é maior que a")
}
```

**Nota:** No R, utilizamos chaves `{ }` para delimitar o escopo (o bloco de código) que deve ser executado quando a condição for atendida.

## O uso do `else if`
Frequentemente, precisamos testar múltiplas condições em sequência. O `else if` permite que o programa verifique uma nova condição caso a anterior tenha falhado.

```r
a <- 33
b <- 33

if (b > a) {
  print("b é maior que a")
} else if (a == b) {
  print("a e b são iguais")
}
```

O R avaliará as condições em ordem. Se `b > a` for falso, ele passará para o `else if` e verificará se `a == b`. Você pode incluir quantos blocos `else if` forem necessários para cobrir todos os cenários lógicos da sua análise.

## O encerramento com `else`
A cláusula `else` atua como uma rede de segurança: ela captura todos os casos que não satisfizeram as condições anteriores. Ela é essencial para garantir que o seu algoritmo sempre tenha um desfecho definido.

```r
a <- 200
b <- 33

if (b > a) {
  print("b é maior que a")
} else if (a == b) {
  print("a e b são iguais")
} else {
  print("a é maior que b")
}
```

Em cenários de ciência de dados, isso é comumente usado para categorizar variáveis ou tratar exceções em fluxos de processamento. Também é possível utilizar o `else` de forma simplificada, sem o `else if`:

```r
a <- 200
b <- 33

if (b > a) {
  print("b é maior que a")
} else {
  print("b não é maior que a")
}
```
