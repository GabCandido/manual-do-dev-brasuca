## Tutorial de Introdução ao R

O **R** é uma linguagem de programação e um ambiente de software desenvolvido especificamente para **computação estatística** e **análise de dados**.

Ao contrário de linguagens de propósito geral (como Python ou C++), o R foi desenhado por estatísticos, para estatísticos. Isso significa que sua sintaxe e bibliotecas nativas são otimizadas para manipular vetores, matrizes e realizar operações matemáticas complexas com poucas linhas de código. No mercado atual, é a ferramenta padrão-ouro em ciência de dados, bioinformática e econometria, sendo essencial para transformar dados brutos em insights visuais e estatísticos.

## Exemplos em Cada Capítulo

Com o nosso editor "Try it Yourself", você pode editar códigos R diretamente no navegador e visualizar o resultado instantaneamente. Isso é fundamental para entender a **reatividade** da linguagem.

### Exemplo: Saída de texto e cálculo simples

O R permite realizar cálculos rápidos de forma intuitiva, funcionando como uma calculadora poderosa e um motor de script ao mesmo tempo.

```r
"Hello World!"
5 + 5
```

**Explicação:** O primeiro comando imprime uma *string* no console. O segundo realiza uma operação aritmética básica. Ao executar, você notará o prefixo `[1]` no resultado; isso indica que o R está retornando o primeiro elemento (e, neste caso, o único) do vetor resultante.

[🚀 Pratique este código](https://www.w3schools.com/r/tryr.asp?filename=demo_default)

### Exemplo: Criação de gráficos

Uma das maiores forças do R é a sua capacidade de visualização. O comando `plot()` é um exemplo da filosofia da linguagem: com um único comando, o R infere o tipo de gráfico mais adequado.

```r
plot(1:10)
```

**Explicação:** Aqui, `1:10` gera uma sequência de números de 1 a 10. A função `plot()` mapeia esses valores automaticamente para os eixos X e Y. Em cenários reais, isso permite que cientistas de dados realizem a Análise Exploratória de Dados (EDA) em segundos.

[🚀 Pratique este código](https://www.w3schools.com/r/tryr.asp?filename=demo_plot)
